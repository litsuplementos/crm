⚙️ Configuración
🔓 Permisos de edición
Permitir edición de registros vendidos
Los agentes podrán editar sus propios registros archivados con estado Vendido.
🏆 Clientes Fieles
Define cuántas unidades acumuladas califican a un cliente como fiel y el descuento que se les aplicará automáticamente.
Al registrar el celular de un cliente fiel en el formulario, el descuento se precargará automáticamente.
🎯 Objetivo de unidades por día
Define cuántas unidades mínimas debe vender cada agente por día (jornada de 9h). Esto activa los emojis animados y el panel de seguimiento.
Ejemplo: 10 unidades → un emoji cae cada 54 minutos durante la jornada.
Emojis animados en todas las pestañas
Desactiva si los emojis que caen resultan distractores.
🗓 Filtro de tiempo por defecto
Define qué período se muestra por defecto al abrir el dashboard y registros.
Se guarda en base de datos y aplica a todos los agentes como valor inicial.
🕐 Horario laboral
Define los turnos de trabajo. Los emojis y el ritmo se calculan sobre este horario.
🌅 Turno Mañana
🌆 Turno Tarde
Los emojis se distribuyen uniformemente a lo largo de la jornada según el horario configurado.
📋 Reglas automáticas
🔁 3× Rellamada → Se archiva automáticamente como No interesado
📵 4× Sin respuesta → Se archiva automáticamente como No interesado
❌ 2× Cancelado → Estado cambia a SPAM automáticamente
⚠️ El admin puede editar manualmente cualquier estado para revertirlo
